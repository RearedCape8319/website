These projects show something with little, if any, user input

Sierpinski Triangle - A dot continuously moves half way to a randomly chosen corner of a triangle, which ends up plotting the Sierpinski Triangle fractal. (No input)

Hitomezashi Stitch Pattern - Dots on a grid are connected at every other interval, offset by either 0 or 1 as predetermined. The offset values are randomised each run. (No input)

Marching Squares - A field of noisily random values will have lines drawn around them to map the edge between high and low values. (No input)

Random Games - A random game is chosen and the corresponding task and game name is given to the user. (Use buttons to interact, scroll through list of games available)

Covid Test Probability - Tells the user how much to trust the covid test. (Change values as needed, use button to recieve result)
